# Nome: Projeto VM Completa
Feito por João Vitor Lavigne

Uma máquina virtual com S.O Ubuntu 24.04 com conexão no modelo bridge.

### Softwares Instalados:
O modelo já possui a instalação automática dos seguintes softwares:
- Vim
- Curl
- Telnet
- Unzip
- Wget
- Net-tools
- Htop
- Nmap

### Usuário:
O modelo também possui um usuário de nome **Joao** criado com permissão sudo, pasta pessoal na home e bash.
